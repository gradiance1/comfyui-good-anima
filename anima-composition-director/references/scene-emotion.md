# 场景、角色与情绪参考

只在多人、群像、互动、属性归属风险、大场景、天气、道具、动作、复杂服装、剪影、故事性、象征物或关系情绪需要明确构图时读取。

成人/特殊视觉构图需求只在用户明确请求时读取 `adult-runtime/_index.md`。

## 多角色与互动

先决定关系，再决定站位。不要把多人图写成角色名 + 一串未归属的发色/服装。

- 关系类型：并列、面对面互动、主角带群体、群众围绕一个焦点。
- 每个角色绑定一个稳定位置：左、中央、右、前景、中景、背景。
- 每个位置保留身份锚点：发色、服装颜色、道具、徽记或剪影。
- 2 人：面对面、并排对比或过肩。
- 3 人：三角站位，主角在最高对比点。
- 4 人以上：按角色/阵营分组，不给每个人同等细节。
- 群众：只描述一个焦点角色和一种群众行为。

关系布局：

| 意图       | 布局                                     |
| ---------- | ---------------------------------------- |
| 对话       | 双人镜头或过肩；视线和嘴部可见           |
| 协作       | 共享道具或共同方向；身体朝同一目标       |
| 冲突       | 两侧分离、相反对角线、中间留负空间       |
| 保护       | 保护者在前景，被保护者在后方或更低       |
| 领袖与群体 | 领袖在最高对比点，群体成弧线或三角       |
| 人群焦点   | 一个命名焦点主体，人群作为纹理或重复手势 |

互动可读性：

- 用视线、手部接触、共享道具或对立姿势表现关系。
- 互动依赖手和脸时，手和脸必须清楚。
- 对话/情绪重要时，用正面、三分之四或过肩，不要让相机自由变成背影队列。
- 相似服装用间距、明暗对比或不同剪影分开。
- 两角色重叠必须有意图：拥抱、抓握、保护、支撑、攻击、耳语。

## 身份与属性归属

- 发色、服装颜色、道具、表情必须绑定到具体角色或位置。
- 未命名群体用位置：左侧角色、中央角色、右侧角色、前景主体、背景二人组。
- 角色很多时，只保留焦点主体完整身份，其他用角色功能描述。

稳定句式：

```text
Place the black-haired girl on the left, holding the lantern.
Keep the blonde center character in front, with the crowd behind her.
Put the taller character in the foreground, shielding the smaller character.
```

## 场景与动作

选一个场景锚点，让所有可见细节支持它：

- 角色动作：奔跑、伸手、转身、守护、表演。
- 道具互动：伞、剑、书、杯、手机、乐器、车辆。
- 天气效果：雨、雪、风、日光、雾、热浪。
- 地点功能：神社台阶、教室课桌、车站站台、集市摊位、厨房台面。

环境构图：

- 角色和场景都重要时，把主体放中景。
- 前景物只在能指向主体时使用。
- 背景线条、道路、窗、货架、街道用于引导视线。
- 脸和手后面的背景保持简单。
- 远景用门、树、建筑、人群作为尺度参照。
- 俯视/航拍要显示可读地面布局：道路、河流、屋顶、田野、阴影、人群或一个地标。
- 天气必须影响可见物：头发、衣服、道具、地面、光或姿势。

场景配方：

| 场景      | 构图计划                                             |
| --------- | ---------------------------------------------------- |
| 雨夜街道  | 主体在中景；伞或湿发可读；店铺灯在后方               |
| 雪地/神社 | 浅色服装后放深色树木、鸟居或建筑；脚印或肩上雪       |
| 风中屋顶  | 沿风向留白；头发/缎带/披风表现风；地平线低于脸       |
| 车站站台  | 站台线引向主体；标识留背景；栏杆不遮脸               |
| 教室窗边  | 主体靠窗；课桌制造纵深；脸受窗光                     |
| 市集/祭典 | 摊位或灯笼框住两侧；焦点角色与人群用对比分开         |
| 森林小径  | 小径或树干引导视线；叶子框边缘；脸部区域干净         |
| 航拍地景  | 地形占主导，一个地标或小主体提供尺度；不期待脸部细节 |

## 动作、姿势、表情、服装一致性

- 描述动作峰值，不描述连续事件。
- 运动或视线方向留出空间。
- 动作道具剪影清楚，并远离脸。
- 表情匹配动作：用力、惊讶、冷静、恐惧、喜悦、紧张。
- 服装匹配天气和动作：雨中湿布、风中飘发、雪中厚衣。
- 承重姿势要有物理线索：落地脚、弯膝、倾斜躯干、抓握手或支撑臂。
- 微妙动作让手、眼睛和道具承载故事。

| 意图      | 姿势/表情                        | 服装/道具互动                  |
| --------- | -------------------------------- | ------------------------------ |
| 平静肖像  | 放松肩膀、稳定视线、小手势       | 服装剪影干净，道具静止         |
| 寻找/惊叹 | 抬头或看画外，手靠近脸/物件      | 头发和轻布料被环境轻微带动     |
| 奔跑/追逐 | 前倾、落地脚、用力表情           | 外套、裙摆、围巾或头发跟随运动 |
| 守护      | 身体挡在威胁和目标之间           | 武器/盾/伞阻挡来路             |
| 表演      | 开放手势、手可读、脸朝向观众或光 | 乐器/麦克风/扇子与手清楚连接   |
| 安静亲密  | 距离近、柔和视线、小动作         | 共享道具、袖子触碰或肩膀重叠   |

## 道具、武器与服装

道具规则：

- 大道具要有负空间，不横跨脸。
- 武器、伞、乐器、书、手机必须看见手部接触。
- 道具定义场景时，角色要使用、携带、倚靠或反应它。
- 长道具沿对角线或背景线摆放，避免杂乱。
- 小道具需要更近镜头，或让手/视线指向它。
- 不添加不影响姿势、视线、动作或场景功能的道具。

| 道具      | 放置                                          |
| --------- | --------------------------------------------- |
| 伞        | 肩上或侧面，随雨/风倾斜，不遮脸               |
| 剑/杖     | 对角线穿过开放空间，握柄可见，尖端不贴头      |
| 书/手机   | 靠近胸前或桌面，眼睛/手指向它                 |
| 乐器      | 手和接触点可见，姿势围绕乐器                  |
| 车辆      | 默认作为尺度/背景锚点，除非用户要求角色在车上 |
| 食物/饮料 | 足够靠近手口关系，表情可读                    |

服装与剪影：

- 复杂服装需要更干净背景，中景或全身保持轮廓。
- 独特身份锚点必须可见：帽子、徽记、领口、袖子、裙形、甲片、缎带。
- 不要在脸周围堆许多小配饰。
- 天气影响服装时，只表现一个清楚效果：湿布、风吹头发、肩上雪、阳光边缘。
- 分层服装先保护外轮廓，再展示一两个关键细节。
- 服装和背景同色时，用轮廓光、深色背景块或负空间分离。

## 故事性与象征物

单帧故事只选一个可见转折：伸手、拒绝、察觉、隐藏、保护、离开、等待、发现。

故事通过视线、手部接触、距离、物件位置、门槛、画外方向表现，不写背景设定。

可用一个前后线索：掉落物、打开的门、断裂缎带、空椅子、消退光线、湿脚印、被扰乱的花。

象征物只用一个主 motif。把抽象概念转成可见物体、动作、关系或站位：花、碎镜、关门、空椅、影子、窗、缎带、伞、鸟、月、水面倒影、伸手、背身、距离、遮挡。

比拟/隐喻要通过位置、颜色呼应、手部接触、视线、同方向光、平行动作或共享剪影连接主体；不要解释隐喻。

## 情绪构图

| 情绪      | 可见构图控制                             |
| --------- | ---------------------------------------- |
| 渴望      | 主体看向画外；视线方向留负空间           |
| 孤独      | 小主体对大而安静的空间；无人接触         |
| 保护      | 保护者挡住威胁线；被保护者在后方或更低   |
| 犹豫      | 手靠近但未触碰人/物；身体背离视线        |
| 悲伤/失去 | 低头、附近空物件、低饱和光、更大阴影面积 |
| 希望      | 脸或手接住光；出口、窗、天空或花更亮     |

负空间用于孤独、等待、压力或沉默时，必须被视线、光或背景形状指向。

## nltags 示例

```text
Place the red-haired girl on the left, holding the shared umbrella.
Keep the silver-haired girl on the right, looking toward her.
Use triangle placement for the three characters around the table.
Keep each character's hair color and outfit clearly separated.
Use a three-quarter two-shot so both faces and eye lines remain visible.

Place her in the midground, running toward the open street.
Frame her with wet storefronts while keeping her face clear.
Let the umbrella tilt with the wind, away from her face.
Use the train platform lines to lead attention toward her.
Show the classroom desks receding behind her toward the window.
Anchor her feet on the wet road with a soft contact shadow.
Use an aerial view with roads and rooftops showing the ground layout.

Keep the umbrella above her shoulder, away from her face.
Show the sword silhouette diagonally with a clear hand grip.
Keep the face visible; place the sword beside the face, not through or across the face area.
Use a clean background so the layered outfit remains readable.
Separate the dark outfit from the night street with a thin rim light.

Place a single wilted flower near her hand, echoing her lowered gaze.
Use an upper-body frame if her facial emotion must remain readable.
Use a hand-and-letter insert only when the prop is more important than facial emotion.
Use the open doorway behind her as the only story clue.
Let her face catch the window light while the room stays subdued.
Keep her hand close to the letter without touching it.
Keep the frame upright and stable unless a tilted composition is requested.
```
