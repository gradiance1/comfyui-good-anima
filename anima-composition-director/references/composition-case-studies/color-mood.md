# Color & Mood / 色彩情绪案例

## 核心认知
色彩不等于「加什么tag」。色彩决定了画面在读者看到任何具体内容之前的第一情感反应。选错色温 = 画面还没看清楚就已经不对劲了。

---

### Pattern 1：暖色情绪（Warm Palette）

**场景→色温对照**
```
温暖的日常/恋爱/告白/重逢 → warm lighting, golden hour, amber glow
战斗中的愤怒/狂暴/绝境爆发 → fiery, blazing, volcanic, intense red
夕阳下的告别/回忆/怀旧 → sunset, golden light, nostalgic warmth
韩式亲密/暧昧/成人氛围 → warm amber, candlelit, intimate orange
```

**Wrong Pattern**
- 暖色 + 恐怖/孤独题材 → 画面情绪混乱
- 暖色过度 → 画面看起来像被泼了橙汁
- 暖光 + 冷色服装/人物 → 色彩打架

**Correct Prompt Control**
```
warm lighting, golden hour, amber glow,
soft warm light, nostalgic atmosphere,
sunset palette, orange tint, cozy
```


### Pattern 2：冷色情绪（Cool Palette）

**场景→色温对照**
```
孤独/内省/失去/回忆 → cool lighting, blue tone, melancholic
夜晚/月光/秘密/密会 → moonlight, night scene, cobalt blue
科技/未来/机械/赛博 → neon blue, cyan light, cool artificial
悬疑/紧张/冻结/静止 → cold atmosphere, steel blue, icy
```

**Wrong Pattern**
- 冷色 + 欢乐庆典 → 像参加了葬礼
- 冷色 + 暖色人物 = 分裂（除非故意对比）

**Correct Prompt Control**
```
cool lighting, blue hour, moonlight,
cold tone, lonely atmosphere, night scene,
moonlit, pale blue, steel blue, icy
```


### Pattern 3：冷暖对比叙事（Warm/Cool Narrative Split）

**Wrong Pattern**
- 冷暖边界模糊 → 看起来像调色失误
- 冷暖比例50:50 → 太均等了，没有主导情绪

**Correct Prompt Control**
```
warm window light, cold exterior,
cozy inside, harsh outside,
split lighting, warm left cold right,
contrast temperature, two worlds
```


### Pattern 4：补色冲击（Complementary Contrast）

**Wrong Pattern**
- 补色饱和度都推满 → 在打架而不是在互补
- 补色面积1:1 → 平分画面，不和谐
- 补色用在需要和谐的日常场景 → 过度刺激

**Correct Prompt Control**
```
complementary colors, orange and blue color scheme,
warm character lighting with cool shadow,
contrasting color palette, cinematic color grade
```
- **关键**: 互补色 ≠ 两个颜色平分，而是一个主色+小面积的互补点缀


### Pattern 5：单色情感深度（Monochromatic Depth）

**Wrong Pattern**
- 完全去色（黑白） → 丢失了色温的叙事功能
- 饱和度全压到零 → 像没上色的线稿
- 单色场景 + 复杂的背景线条 → 层次混乱

**Correct Prompt Control**
```
monochrome, limited palette,
desaturated colors, muted tones,
blue monotone, gold accent,
single color mood, restrained palette
```

## 色彩速查表

| 你想传达的情绪 | 主色 | 搭配 | 对应tag |
|:-------------|:----|:-----|:------|
| 温馨/日常/爱情 | 暖橙/琥珀 | 奶油白、淡粉 | `warm lighting, golden hour, soft amber` |
| 孤独/回忆/忧伤 | 冷蓝/青 | 淡灰、紫 | `cool lighting, blue tone, melancholic` |
| 愤怒/战斗/狂热 | 深红/热橙 | 黑、暗紫 | `fiery, blazing, intense red, volcanic` |
| 清新/治愈/自然 | 浅绿/薄荷 | 白、淡黄 | `fresh green, mint, soft sunlight` |
| 神秘/魔法/梦幻 | 紫/品红 | 暗蓝、金 | `magical, purple glow, ethereal` |
| 紧张/悬疑/冷静 | 钢蓝/灰 | 白、黑 | `cold steel blue, tense, sharp contrast` |
| 爱/暧昧/亲密 | 玫瑰/桃红 | 奶油、橙 | `rosy, pink hue, romantic flush` |
