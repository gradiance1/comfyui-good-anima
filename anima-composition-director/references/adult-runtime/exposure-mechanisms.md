# Exposure Mechanisms Reference / 露出服装机关参考

> **T3 Runtime 参考** — 只在需要精确控制服装暴露方式/机关设计/场景化露出时读取。
> 不覆盖服装轮廓设计（→ clothing-silhouette-reference.md）。

---

## 一、面料机关六型

| 类型 | 机制 | 视觉特征 |
|------|------|---------|
| 透明型 | 微透→中透→全透三级 | 底层皮肤/内衣透过布料可见，透明度渐进 |
| 光感变透 | 室内正常，强光下透视 | 逆光/聚光灯下突然暴露，正常光下不透明 |
| 遇水变透 | 湿度触发透明 | 雨淋/泼水/出汗后局部透明，干后恢复 |
| 局部透明 | 指定部位透视 | 仅胸部/臀部/胯部区域透明，其余正常 |
| 镂空型 | 圆形/菱形/心形开洞 | 规则形状的暴露窗口，边缘可有蕾丝/金属环 |
| 多层遮蔽 | 外层正常/内层暴露 | 外套正常→脱掉或敞开→内部暴露 |

**nltags 写法**：`Her dress turned translucent under the spotlight, the outline of her body suddenly visible through the once-opaque fabric.`

## 二、物理机关四型

| 类型 | 触发机制 | 视觉结果 |
|------|---------|---------|
| 磁扣型 | 动作触发自动分离 | 特定姿势下衣物突然分开 |
| 拉链型 | 隐藏拉链控制暴露范围 | 拉链开口=暴露窗口大小 |
| 暗扣型 | 一拉全开 | 单点触发→整件散开 |
| 动作触发 | 弯腰/举手/坐下/蹲下/跨步 | 日常动作导致意外走光 |

**nltags 写法**：`The hidden zipper traced down her back. One pull would expose everything beneath.`

## 三、服装部位露出细分

| 服装类型 | 机关设计要点 |
|---------|------------|
| 上衣 | 领口深度(锁骨→胸线)、腋下镂空、后背全开、纽扣间距 |
| 半身裙 | 侧开叉高度、前开叉、后开叉、腰线下移 |
| 连衣裙 | 胸线开口+腿侧开叉双机关，背后拉链长度 |
| 短裙 | 裙摆长度(膝上→大腿根)、风掀/弯腰走光角度 |
| 泳装 | 系带式(一拉散开)、侧开口、高叉裁剪 |
| 内衣 | 前扣/后扣、可拆卸肩带、透明材质、无衬垫 |
| 制服 | 纽扣间距、裙长修改、内搭省略 |

## 四、环境场景化露出规则

| 场景 | 视觉条件 | 曝光方式 |
|------|---------|---------|
| 夜间街道 | 黑暗掩护+路灯间歇暴露 | 路灯下突然可见，阴影中隐匿 |
| 公共交通 | 车厢晃动掩护 | 刹车/转弯时衣服位移走光 |
| 商场/试衣间 | 镜子反射+帘缝 | 对镜自拍、帘子未拉紧 |
| 公园 | 树丛+长椅 | 自然遮蔽→偶然路过者视角 |
| 窗边 | 剪影+对面楼 | 背光剪影→开灯后暴露 |
| 办公室 | 桌下空间 | 上半身正常→下半身隐于桌下 |
| 海滩 | 泳装合理化 | 比基尼系带松脱 |
