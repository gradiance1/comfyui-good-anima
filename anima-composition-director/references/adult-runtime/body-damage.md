## 一、SP（打屁股）伤痕 5 级颜色光谱

从轻到重的递进光谱。每级选择 1 个层级，不要跨越 2 级以上。

| 等级 | 颜色     | 光泽      | 触感/视觉                      |
| :--- | :------- | :-------- | :----------------------------- |
| Lv1  | 粉红色   | 哑光      | 轻微泛红，与周围肤色自然过渡   |
| Lv2  | 鲜红色   | 微光      | 明显红色块，边缘模糊           |
| Lv3  | 深红肿胀 | 油光      | 皮肤紧绷发亮，浮肿形成硬棱     |
| Lv4  | 紫红交织 | 油光+淤青 | 红紫交错，硬棱子凸起，深层淤血 |
| Lv5  | 紫黑破坏 | 湿光      | 表皮破损渗液，紫黑淤血扩散     |

**nltags 写法**：选词要精确——`flushed pink`(Lv1) → `bright red handprint`(Lv2) → `swollen deep red welts`(Lv3) → `purple-red bruises with raised edges`(Lv4) → `broken skin seeping fluid, dark purple-black bruising`(Lv5)

## 二、工具 × 皮肤痕迹几何对应

| 工具      | 痕迹形态           | 几何特征                       |
| --------- | ------------------ | ------------------------------ |
| 藤条      | 细红线肿棱         | 鞭梢弯曲包覆形成弧线，宽 2-3mm |
| 木板/戒尺 | 长方形边缘清晰红印 | 平行边缘 + 气孔圆点（木纹印）  |
| 皮带      | 宽带状柔和淤青     | 边缘模糊，宽度 = 皮带宽        |
| 竹板      | 整齐排列条印       | 等距平行线，竹节处加深为斑点   |
| 数据线    | 极细切割痛痕       | 线宽 < 1mm，可形成交叉网格     |
| 手印      | 五指+掌形          | 指缝间隙无痕，掌心实红         |

**nltags 写法**：`thin red welts crisscrossing her skin`（藤条）、`rectangular red marks with visible wood grain pattern`（木板）、`broad belt-shaped bruises fading at the edges`（皮带）

## 四、服装 × 击打物理反馈

不同服装材质对击打的缓冲和痕迹表现不同：

| 服装       | 缓冲 | 视觉表现                               |
| ---------- | :--: | -------------------------------------- |
| 芭蕾连体衣 |  无  | 紧绷清脆，痕迹直接透过薄布显现         |
| 丝绸       |  低  | 工具加速滑动，冰火温差感，布料本身无痕 |
| 棉内裤     |  中  | 湿润后半透明，透出底层红色             |
| 渔网袜     |  零  | 网格状挤压淤痕——网格压入皮肤的几何烙印 |
| 真空丝袜   | 极低 | 汗液形成油亮高光，红痕透过丝袜可见     |

**nltags 写法**：`The fishnet pattern pressed deep into her skin, each diamond leaving its own red welt.` / `Through the soaked cotton, the deep red marks bloomed like watercolor.`

## 六、妆容毁灭分层视觉

从有序精致 → 被欲望涂毁的多层过程：

1. **底妆融化**：汗水+体温 → 斑驳油性质感条痕，毛孔处积聚油珠
2. **眼妆溶解**：泪水冲开睫毛膏和眼线 → 脸颊两道黑色"泪痕"
3. **眼影泥浆**：彩色粉末变成糊状堆积在眼窝
4. **唇印乳化**：蜡质唇印接触温热液体 → 色彩溶解扩散，与乳白混合成浑浊乳液

**nltags 写法**：`Mascara ran in two dark rivers down her cheeks. Her carefully painted lips were now a smeared, glossy mess.`

## 十、镜子/眼罩感官机制

- **镜子**：受罚者实时看到自身伤痕变色+肌肉颤动 → 羞耻正反馈 → 公开处刑错觉
- **眼罩**：视觉剥夺 → 听觉触觉代偿放大 → 落点不可预测 → 时间感模糊 → 面部表情完全暴露
- **nltags 写法**：`She watched herself in the mirror, every new red mark blooming across her reflection.`
