## 2. 日式色气构图技巧

### 仰视·俯视构图

- **仰视（Low Angle）**: 从下往上看。让角色显得更大更有支配感
  - 胯间仰视 → 脸+局部同时展示的最佳选择
  - 仰卧角色从脚端仰视 → 开腿被强调
- **俯视（High Angle）**: 从上往下看。让角色显得更小更弱
  - 传教士体位从上方俯视 → 全身交缠一览无余
  - 从床正上方看 → 两人位置关系一目了然

### 透视强化 (Perspective Exaggeration)

- **夸张透视法**: 近处部位（手·脚·胸）放大，远处缩小
  - 传教士位把脚放近处 → 脚底大、脸小
  - 骑乘位把胸放近处 → 胸被强调同时脸也可见
- **鱼眼镜头效果**: 画面中心膨胀。适合收纳整个房间+床上的两人
- **消失点强化**: 强调纵深线，把视线拉入画面

### 视线引导 (Eye Flow)

- 脸 → 胸 → 局部 → 腿，按顺序引导视线是基本构图
- S 形曲线: 头 → 肩 → 腰 → 臀 → 腿的流线
- 对角线: 左上(脸)→ 右下(局部) 是最自然的视线引导
- 视线引导相关 tag: `looking at viewer`, `eye contact`, `guided line`

### 透过·半透明技巧

- 湿透的布料 → 透出乳头·局部的剪影
- 网袜·丝袜 → 强调腿线同时遮挡
- 蕾丝材质 → 阴影复杂增加立体感
- 水滴·汗 → 增加透明感和光泽

## 3. 韩式 R19 构图技巧

### 韩式 R19 三大特征

1. **精美角色设计**: 角色美感最优先。无论什么体位脸都不能崩
2. **故事驱动**: 性行为也有上下文。「为什么这两人现在在做」很明确
3. **阶段性暴露**: 重视一件件脱衣服的过程。不会一下子全裸

## 4. 本子式分镜·展开技巧

### 场景展开的黄金模式

1. **导入·日常** → 2. **诱因（某个契机）** → 3. **前戏（脱衣）** → 4. **插入（最初的合体感）** → 5. **发展（变换体位）** → 6. **高潮（表情+体液）** → 7. **余韵（事后处理）**

### 各阶段的构图要点

| 阶段 | 构图技巧                                            |
| :--- | :-------------------------------------------------- |
| 导入 | 视线交汇、手触碰、靠近。背景展示两人的距离          |
| 前戏 | 特写多。口 → 手 → 胸 → 局部依次对焦。衣服一件件减少 |
| 插入 | 正面或正侧最能传达。面部表情很重要                  |
| 发展 | 展示体位变换的瞬间。让人感受腰部动态的线条          |
| 高潮 | 脸+体液+局部的同框展示。背景简单或崩溃化            |
| 余韵 | 肩部喘息·湿润肌肤·凌乱头发·迷离眼神                 |

### 本子式「留白」技巧

- **「跨页」一图**: 跨页 2 页做成一张大格。冲击力极强
- **大小对比**: 大格(情况说明)和小格(细节·表情)的节奏感
- **视线一致**: 格内角色视线与下一格位置联动
- **「留白」**: 在行为间隙插入日常镜头，反而增加色气

## 7. R15-R17「不展示的构图」黄金法则

> 具体 10 技法详见 `composition-case-studies/environment-storytelling.md` Pattern 9

1. **用一张画让人想象「前后故事」**

   - 脱到一半的衣服 → 「接下来要脱了」的想象
   - 凌乱的床 + 湿润肌肤 → 「已经结束了」的想象
   - 脸红舔唇 → 「想要了」的想象

2. **「若隐若现」最强**

   - 完全隐藏不色。完全展示是 R18
   - 乳头从发丝缝隙中隐约可见。乳晕刚好被遮住
   - 局部被布覆盖但轮廓浮出
   - 手在遮挡但指缝中窥见

3. **表情说明一切**

   - R15-R17 中正因为看不到局部，面部表情最重要
   - 湿润的眼·微张的嘴·泛红的脸颊·细密的汗 → 用这些表达一切
   - `ahegao` 是 R18。R15-R17 用 `flushed, wet eyes, parted lips, heavy breathing`

4. **用水·汗·湿润表现「正在此刻」**
   - 被水淋湿后的头发和衬衫 → 色气 3 倍
   - 汗水反光的肌肤 → 联想到运动中/刚结束
   - 光泽的嘴唇 → 联想到接吻/口渡
