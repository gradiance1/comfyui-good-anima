# Adult / Special Visual Runtime Index

> 仅在用户明确请求成人向或特殊视觉状态，并且当前运行环境允许处理该类内容时读取。
> 普通构图规划、普通失败复盘、普通服装/材质问题不得进入本索引。

## 边界

- 不从 `../composition-case-studies/_index.md` 自动跳转到这里。
- 不把本索引当通用构图参考；只处理用户明确提出的专项视觉稳定性问题。
- 每次最多读取 1 个最相关文件；读取后只提取必要的结构保护和冲突消解规则。
- 如果用户需求不明确，回到普通构图或服装参考，不主动扩展成人内容。
- 特殊主题类（NTR/BDSM/调教/胁迫等）需要跨槽位协调时，读 `special-themes.md` 获取配方。
